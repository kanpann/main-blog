# Key Promoter X
[Key Promoter X](https://plugins.jetbrains.com/plugin/9792-key-promoter-x)
![](https://plugins.jetbrains.com/files/9792/screenshot_17105.png)
마우스로 단축키가 지정되어 있는 탭 같은 곳을 클릭했을 때 그 단축키를 팝업으로 알려주는 플러그인. 솔직히 뜨는 게 신경 쓰이긴 하지만 자주 쓰는 키를 모두 외운 후 삭제하면 되므로 추천한다.

# Lombok
[Lombok](https://plugins.jetbrains.com/plugin/6317-lombok/)    
사람들이 Java를 기피하게 만든 원인. 매번 필수적으로 작성하는 getter, setter 등을 작성하지 않도록 지원해 준다. 이 플러그인은 요즘엔 인텔리제이를 깔면 자동으로 깔려있을 정도로 필수라고 볼 수 있다.

# Rainbow Brackets
[Rainbow Brackets](https://plugins.jetbrains.com/plugin/10080-rainbow-brackets)
![이미지](https://plugins.jetbrains.com/files/10080/screenshot_17373.png) 괄호, 중괄호별로 색깔을 다르게 주어 구분하기 편하게 도와준다.

# Advanced Java Folding
[Advanced Java Folding](https://plugins.jetbrains.com/plugin/9320-advanced-java-folding)
![image](https://user-images.githubusercontent.com/45007556/124851144-ebfda580-dfdc-11eb-80ed-ef261b28f784.png) 자바의 장황함을 개선한 최신 JVM언어(코틀린, 스칼라)들이 있다. 이 플러그인을 쓴다면 그 언어들과 비슷하게 간결하게 보여주어 조금 더 집중해야 할 곳에 집중할 수 있게 만들어준다.(실제 코드가 수정되지 않음)

# .ignore
[.ignore](https://plugins.jetbrains.com/plugin/7495--ignore)
![](https://plugins.jetbrains.com/files/7495/screenshot_14958.png) 보통 git과 같은 곳에 올라가지 말아야 할 파일이 있을 때 ignore를 직접 생성하여 작성해 준다. 허나 이 플러그인을 쓰게 되면 쉽게 ignore를 작성할 수 있다.

# Translator
[Translator](https://plugins.jetbrains.com/plugin/9690-translator)
![](https://plugins.jetbrains.com/files/9690/screenshot_17011.png)
가끔 메소드나 변수 이름을 지을 때 영단어가 생각이 안 날 때가 있다. 이 플러그인은 그런 상황에서 매우 혁신적이다..!

# CodeGlance2
[CodeGlance2](https://plugins.jetbrains.com/plugin/15562-codeglance2)
<img width="400" src="https://user-images.githubusercontent.com/45007556/124854589-82809580-dfe2-11eb-921b-4a321677d9c3.png"/>
인텔리제이에서 vscode처럼 우측에 코드 미니맵을 표시해 준다.

# Prettier
[Prettier](https://plugins.jetbrains.com/plugin/10456-prettier)  
이 플러그인은 JS 기반 프론트 개발할 때 유용한 플러그인인데, 정해진 코딩 스타일에 맞게 코드를 자동으로 변환해 준다. 만약 프론트 개발을 인텔리제이로 한다면 필수라고 생각한다.
